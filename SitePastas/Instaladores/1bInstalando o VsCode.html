Instalando o VsCode


💻 Passo a passo para instalar o VS Code no seu computador
✅ 1. Acesse o site oficial:

🔗 https://code.visualstudio.com/

Clique em Download for Windows (ou escolha macOS/Linux, se for o seu caso).
✅ 2. Instale o VS Code
No Windows:

    Dê duplo clique no arquivo .exe que foi baixado.

    Marque a opção "I accept the agreement".

    Clique em Next até chegar na tela "Select Additional Tasks".

    Marque:

        ✅ "Add to PATH" (muito importante!)

        ✅ "Register Code as an editor"

    Clique em Install.

    Finalize com "Launch VS Code".

✅ 3. Instale a extensão do Python (importante!)

Assim que o VS Code abrir pela primeira vez:

    Clique no ícone de extensões na barra lateral esquerda (parece um quadradinho).

    Pesquise por Python.

    Instale a extensão "Python – by Microsoft" (mais de 80 milhões de downloads).

✅ 4. Teste se está funcionando com Python

    Crie um novo arquivo: teste.py

    Digite:

print("Olá, Python!")

    Clique com o botão direito no arquivo e selecione "Run Python File in Terminal" (ou use o atalho Ctrl + F5).

Se aparecer “Olá, Python!” no terminal abaixo, está tudo funcionando ✅
💡 Dicas úteis

    Atalho para abrir o terminal: Ctrl + aspas (```) ou Ctrl + J

    Para instalar pacotes: use pip install nome_do_pacote no terminal

⬅ Voltar para o menu
