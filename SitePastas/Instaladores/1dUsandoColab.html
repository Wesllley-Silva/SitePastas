Usando o Colab

✅ O que é o Google Colab?

Um ambiente Jupyter Notebook online
Totalmente gratuito
Armazena seus arquivos no Google Drive
Ideal para quem está começando ou trabalha com ciência de dados, automações, etc.

🚀 Como usar o Google Colab (passo a passo)

🟢 1. Acesse o Google Colab
🔗 https://colab.research.google.com
(É necessário estar logado com uma conta do Google)

🟢 2. Criar um novo notebook
Clique em "Novo notebook"
Ele criará um arquivo com extensão .ipynb

🟢 3. Escrevendo e executando código
Cada célula é um bloco de código Python

Exemplo:
print("Olá, Google Colab!")

Pressione Shift + Enter para executar a célula ou clique no botão de play ▶️

🟢 4. Criar novas células
Clique em + Código ou + Texto (Markdown)

🟢 5. Salvar no Google Drive
Salvamento automático em "Colab Notebooks" no Drive
Ou vá em Arquivo > Salvar uma cópia no Drive

🟢 6. Instalar pacotes como no terminal:
!pip install pandas
!pip install selenium

🧠 Exemplo prático:
# Soma simples
a = 5
b = 3
print("Soma:", a + b)
