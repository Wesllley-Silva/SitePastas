Instalando o Jupyter


💻 1. Acesse o site oficial

Vá para o site oficial do Anaconda:

🔗 https://www.anaconda.com/download
🧩 2. Escolha a versão para seu sistema operacional

Selecione o seu sistema:

    Windows: clique em “Download” do instalador .exe (recomendado 64-bit)

    Mac: baixe o .pkg ou .sh

    Linux: baixe o .sh apropriado

📥 3. Instale o Anaconda (exemplo no Windows)

    Dê duplo clique no arquivo baixado

    Clique em “Next” até chegar na tela de instalação

    Aceite os termos e prossiga

    Escolha “Just Me” (instala só para o seu usuário)

    Marque a opção “Add Anaconda to my PATH” (opcional, mas útil)

    Clique em “Install”

    Ao final, clique em “Finish”

🧪 4. Testar se funcionou
🟢 Abrir o Anaconda Navigator

    Procure no menu iniciar por “Jupyter Notebook”

    Depois é so instalar o que for usar em cada código

⬅ Voltar para o menu
