O que aprendi

O que são Variáveis?
Variáveis são nomes que armazenam valores na memória do computador. Elas servem para guardar dados que podem ser usados e manipulados ao longo do programa. Pense em uma variável como uma caixa com uma etiqueta (o nome) que guarda um determinado conteúdo (o valor).

nome = "Lucas"  # nome é a variável; "Lucas" é o valor armazenado
  • 1. Ordem de Execução dos Programas

    • Python executa os comandos linha por linha, de cima para baixo, de forma sequencial. Portanto, uma variável precisa ser declarada antes de ser usada, ou o programa gerará um erro.

  • 2. Tipos de Variáveis

    • Python reconhece automaticamente o tipo de dado atribuído a uma variável. Veja os tipos mais comuns:

    Tipo Nome Exemplos
    int Inteiros 10, -3, 1000
    float Decimais 3.14, -0.5, 100.0
    str Texto (string) "Olá", 'Python'
    bool Lógicos True, False 
    idade = 20        # int
preco = 19.90     # float
nome = "Ana"      # str
ativo = True      # bool
  • 3. Misturando Tipos de Variáveis

    • Você pode usar diferentes tipos de variáveis juntos, mas precisa cuidar da compatibilidade:

    # print("Idade: " + 25)  # Erro!
print("Idade: " + str(25))  # Correto

    Principais funções de conversão:

    Função Converte para Exemplo
    int() Inteiro int("10") → 10
    float() Decimal float("3.5") → 3.5
    str() Texto str(25) → "25"
    bool() Booleano bool(0) → False
    bool("abc") → True
  • 4. Comando format

    • O método .format() é usado para inserir variáveis dentro de strings:

    nome = "Carlos"
idade = 30

print("Nome: {}, Idade: {}".format(nome, idade))
print("Nome: {0}, Idade: {1}".format(nome, idade))
print("Nome: {n}, Idade: {i}".format(n=nome, i=idade))
  • 5. F-strings e Mudança de Tipos de Variáveis

    • F-strings são uma forma moderna e prática de formatar texto no Python:

    nome = "Maria"
idade = 25

print(f"Nome: {nome}, Idade: {idade}")
print(f"Idade no próximo ano: {idade + 1}")
  • 6. Lista de Exercícios - Estruturas Sequenciais

    • Exercícios típicos para treinar variáveis envolvem:

    • Ler dados com input()
    • Converter tipos (int(), float())
    • Realizar cálculos
    • Mostrar resultados com print(), format() ou f-strings

    Exemplo de exercício:

    nome = input("Digite seu nome: ")
nota1 = float(input("Digite a primeira nota: "))
nota2 = float(input("Digite a segunda nota: "))

media = (nota1 + nota2) / 2
print(f"{nome}, sua média é {media:.2f}")

Exemplo de código

# Cálculo simples de salário com desconto
salario_bruto = float(input("Digite seu salário bruto: "))
desconto = salario_bruto * 0.1
salario_liquido = salario_bruto - desconto

print(f"Desconto: R${desconto:.2f}")
print(f"Salário líquido: R${salario_liquido:.2f}")
