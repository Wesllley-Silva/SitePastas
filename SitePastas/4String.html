Módulo 2 - Strings em Python

O que aprendi

✅ O que são Strings?
Strings são **sequências de caracteres** usadas para representar textos em Python. Tudo que estiver entre aspas simples `'texto'`, aspas duplas `"texto"`, ou aspas triplas será considerado uma string. ---
  • 1. Criando Strings

    • Você pode criar uma string com:

    mensagem = "Olá, mundo!"
nome = 'Python'
texto_longo = '''Este é um
texto com múltiplas
linhas.'''
  • 2. Índices e Fatiamento (Slicing)

    • Cada caractere de uma string tem uma posição (índice), começando do zero.

    palavra = "Programar"
print(palavra[0])    # P
print(palavra[4])    # a
print(palavra[-1])   # r (último caractere)

    Você também pode pegar pedaços da string com fatiamento (slicing):

    print(palavra[0:5])  # Progr → do índice 0 até o 4
print(palavra[:4])   # Prog → do início até o 3
print(palavra[4:])   # amar → do 4 até o final
print(palavra[::2])  # Pormr → de 2 em 2
print(palavra[::-1]) # ramargorP → string invertida
  • 3. Métodos e Funções de String

    • Strings têm diversos métodos úteis. Eles servem para modificar, analisar ou transformar textos:

    Método Exemplo de Código Resultado/Explicação
    .upper() texto.upper() " OLÁ, MUNDO PYTHON! " (converte para maiúsculas)
    .lower() texto.lower() " olá, mundo python! " (converte para minúsculas)
    .strip() texto.strip() "Olá, Mundo Python!" (remove espaços do início e fim)
    .replace() texto.replace("Python", "Java") " Olá, Mundo Java! " (substitui parte da string)
    .title() texto.title() " Olá, Mundo Python! " (primeira letra de cada palavra em maiúscula)
    .count() texto.count("o") Conta quantas vezes um caractere/substring aparece
    len() len(texto) Retorna o tamanho da string (incluindo espaços)
  • 4. F-Strings e Format - Formatando Strings

    • F-strings são a maneira mais moderna de formatar texto com variáveis:

    nome = "Carlos"
idade = 30
print(f"Meu nome é {nome} e tenho {idade} anos.")

    Formatando números e textos:

    valor = 9.8765
print(f"Valor formatado: {valor:.2f}")  # 9.88

    Formatando como moeda (estilo brasileiro):

    preco = 3499.9
print(f"R${preco:,.2f}".replace(",", "v").replace(".", ",").replace("v", "."))
# R$3.499,90

    Datas e horas:

    from datetime import datetime
agora = datetime.now()
print(f"Hoje é {agora:%d/%m/%Y}")
print(f"Hora atual: {agora:%H:%M:%S}")
  • 5. Alinhamento e Exibição
    • produto = "Arroz"
print(f"{produto:<10} - Alinhado à esquerda")
print(f"{produto:>10} - Alinhado à direita")
print(f"{produto:^10} - Centralizado")
  • 6. Verificações com in e Comparações

    • Podemos verificar se uma string contém outra:

    frase = "Aprendendo Python"
print("Python" in frase)      # True
print("Java" in frase)        # False
print("Python" not in frase)  # False

    Verificando início e fim:

    print(frase.startswith("Apre"))  # True
print(frase.endswith("Python"))  # True

    Comparações:

    nome1 = "João"
nome2 = "joão"
print(nome1 == nome2)            # False
print(nome1.lower() == nome2.lower())  # True
  • 7. Exercícios práticos

    • Exemplo 1: Nome formatado e saudação

    nome = input("Digite seu nome: ").strip().title()
print(f"Seja bem-vindo(a), {nome}!")

    Exemplo 2: Validação de e-mail

    email = input("Digite seu e-mail: ")
if "@" in email and "." in email:
    print("E-mail válido")
else:
    print("E-mail inválido")

    Exemplo 3: Contar vogais

    texto = input("Digite uma frase: ").lower()
vogais = "aeiou"
contador = sum(1 for letra in texto if letra in vogais)
print(f"A frase tem {contador} vogais.")
